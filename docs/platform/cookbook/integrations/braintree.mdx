## Overview

In this tutorial we will look integrate with Braintree using their **hosted fields** and their **GraphQl** endpoint.

We will be focussing soley on Card Payments in the checkout.


## How it works

### Data Flow

1. <ServerFlag /> Configuration data (API keys etc)

2. <ClientFlag /> Braintree form initialised

3. <ClientFlag /> User inputs card data

4. <BraintreeFlag /> `nonce` token grabbed from Braintree

5. <ServerFlag /> Payment query created (using token)

6. <BraintreeFlag /> Payment handled

6. <ServerFlag /> Payment status returned
